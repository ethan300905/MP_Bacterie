--------- Help ----------------
CC : current control (in red)
MW : mouse wheel
-------------------------------
Space	: toggle pause
D	: toggle debug mode
R	: resets simulation
MW	: Zoom
Esc	: close window
C	: refresh config
Arrows	: move simulation
	view
Tab	: switch to next
	control
PageUp/X    : increase CC
PageDown/Y  : decrease CC
M	  : add Monotrichous
P	: add PilusMediated
	bacterium at MP
1-2	: add GroupMotility
          bacterium
	  of type 1-2 at MP
