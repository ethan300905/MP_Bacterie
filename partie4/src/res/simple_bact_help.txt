--------- Help ----------------
CC : current control (in red)
MW : mouse wheel
MP : mouse position
-------------------------------
Space	: toggle pause
D	: toggle debug mode
R	: resets simulation
MW	: Zoom
Esc	: close window
C	: refresh config
Arrows	: move simulation
	view
Tab/Q	: switch to next
	control
PageUp/X    : increase CC
PageDown/Y  : decrease CC
M	  : add Monotrichous
            bacterium
N	: add NutrientB at MP
Shift-N	: addNutrientA at MP
